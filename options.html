Contact Center Extension

Configure CSS selectors and timing parameters

CSS Selectors

These selectors are used to identify elements on your CRM page. Use standard CSS selector syntax.

CSS selector for the element that contains the disposition screen or modal. This element should be hidden/removed when the disposition is complete.
CSS selector for the element that displays the current lead ID. The extension will extract the lead ID from this element's href attribute.
CSS selector for the "New Lead" button. This button will be clicked if a new lead doesn't load automatically.

Timing Parameters (milliseconds)

Time to wait after disposition screen closure before checking the lead field. This allows the DOM to settle. Default: 50ms
Time to wait before checking if the lead ID has changed. If the lead hasn't changed after this delay, the "New Lead" button will be clicked. Default: 2000ms
